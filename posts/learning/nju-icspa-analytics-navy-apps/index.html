Navy-Apps 提供一系列运行在 Nanos-lite 上的用户程序，也包含了 Newlib 库。

libs/

libc/

Newlib 项目, 它是一个专门为嵌入式系统提供的 C 库, 库中的函数对运行时环境的要求极低. 这对Nanos-lite来说是非常友好的, 我们不需要为了配合C库而在Nanos-lite中实现额外的功能. 用户程序的入口位于 navy-apps/libs/libc/start.c 中的 _start() 函数, 它会调用用户程序的 main() 函数, 从 main() 函数返回后会调用 exit() 结束运行.

libos/src/nanos.c

系统调用的用户层封装。

函数 syscall

进行系统调用的接口。

intptr_t _syscall_(int type, intptr_t a0, intptr_t a1, intptr_t a2) {
  int ret;
  asm volatile("int $0x80": "=a"(ret): "a"(type), "b"(a0), "c"(a1), "d"(a2));
  return ret;
}
