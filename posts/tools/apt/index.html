APT - Advanced Packaging Tool

Sep 19, 2018

基本信息

  • 介绍：apt，Advanced Packaging Tool 是Linux下的一款安装包管理工具。常用于 Debian, Ubuntu 等系统。是 dpkg 的前端。

常用命令

很多 apt 命令需要管理员权限。

更新数据库

安装软件前常需更新软件包数据库，以保证信息是最新的。

apt update

查找

apt search <seach name>

查看

apt show <package_name> # 查看package内容

apt list # 查看软件包列表
list 选项描述
--upgradeable可升级
--installed已安装
--all系统中所有能找到的package
-versions所有版本

安装

apt install <package_name1> <package_name2> ...
apt reinstall <package_name> # 重新安装包
选项描述
--no-upgrade安装该package但是不升级
--only-upgrade升级该package
=<version_number>指定版本

卸载

apt remove <package_name> # 移除该package
apt purge <package_name> # 彻底删除该package（包含配置文件）

apt autoremove # 自动移除部分无用软件包
remove 选项描述
--purge删除包，包括删除配置文件

更新

apt upgrade

下载

apt download <package_name>

查看缓存中的所有包名

apt-cache pkgnames
